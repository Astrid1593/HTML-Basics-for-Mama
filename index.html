Titles, text and labels


To have text in a website and be able to style it, you need to have tags that open text. Between the opening and closing tags, you can type what you want the text on the label to be.

Size of the text

h1 tags are the largest, and it goes all the way down to h6. p tags are larger than h4, h5 and h6 tags but smaller than the first 3. The default font for the text is Times New Roman, but you can change it in CSS.

Images

img is short for images on the HTML world. That is what you type for an image tag! To make an image tag, you have to open the tag and type "img". Once you tag, go inside the tag and press space. Then type src="". Search for the picture you want and right click on it. Click on the option that says "copy image address". Now go back to the code and paste the image address inside the quote marks.

Anchors

Anchors can be used as shortcuts to other part of the website. To make an anchor, make a ul tag. Ul is sort of a list. Inside the ul, there have to be li. Those are the items in the ul.
How to make: Make the li inside the ul, and between the tags make an anchor tag (a) Inside the anchor tag, type in where the anchor will take you once you click on it (if it's in your website, type in the the ID of what you want the anchor to take you to).
